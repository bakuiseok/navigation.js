navigation.js

"suk9206/navigation.js" is copies the skip break that matches the form set in "suk9206/navigation.js" from the specified element to the desired element.

The navigation.js uses OL list tags. Because it is to keep the order of LIST HEADING strictly.

  1. LIST HEADING 1

    _

    1. LIST HEADING 1-1

      _

    2. LIST HEADING 1-2

      _

      1. LIST HEADING 1-2-1

        Heading H2 is descendent of LIST HEADING 1-2-1

        Heading H3 is descendent of LIST HEADING 1-2-1

        Heading H4 is descendent of LIST HEADING 1-2-1

        Heading H5 is descendent of LIST HEADING 1-2-1
        Heading H6 is descendent of LIST HEADING 1-2-1

        It descendents of LIST HEADING.

        The navigation.js navigates and copies the incidental descendent of LIST HEADING. As it default set is H2, H3, H4, H5, H6.

        ! You can add or modify element to this collection. Please refer this project github 'README.md'.

      2. LIST HEADING 1-2-2

        _

  2. LIST HEADING 2

    _

    1. LIST HEADING 2-1 *Do not navigation.

      Because this list does not enough a condition as navigation condition.

      The navigation.js condition default set element relationship is OL ∋ LI ∋ H2 - HR.

      ! You can modify set of navigation condition. Please refer this project github 'README.md'.

      > Omit HR tag.
